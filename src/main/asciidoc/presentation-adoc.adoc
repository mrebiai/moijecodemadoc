=== Le "javac"
[source]
$ asciidoctor monfichier.adoc

Génération d'un fichier monfichier.html (html5 par défaut)

[source]
$ asciidoctor -D <output> -T <template> -r <requirements> monfichier.adoc

NOTE: la gem asciidoctor doit être installée
